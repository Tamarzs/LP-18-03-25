1. Repetição controlada

Quando sabemos a quantidade de repetições podemos usar um contador: 

int contador = 1;
int fim = 5;

while( contador <= fim ) {
  printf("%d ", contador);
  contador = contador + 1;
}

1.2. Operadores de atribuição

Podemos atribuir um valor à uma variável em C usando =: 

x = 2;

O valor que existia em x é perdido, o valor 2 fica em seu lugar.

Algumas operações de atribuição podem ser abreviadas: 

x = x + 3;

Pode ser substituído por: 

x += 3;

Outros operadores:

Operação Operador Como fica
x = x + 2; += x += 2;
y = y - 3; -= x -= 3;
z = z * 4; *= x *= 4;
w = w / 5; /= x += 5;
k = k % 6; %= x %= 6; 
int contador = 1;
int fim = 5;

while( contador <= fim ) {
  printf("%d ", contador);
  contador += 1;
}

1.3. Repetição com sentinela

Se não sabemos a quantidade de vezes que devemos repetir operações, usamos um determinado valor de controle para encerrar a repetição: 

#include <stdio.h>

int main(void) {
  int soma = 0;

  int numero;

  printf("Digite um número (-1 para encerrar): ");
  scanf("%d", &numero);

  while(numero != -1) {
    soma = soma + numero;

    printf("Digite um número (-1 para encerrar): ");
    scanf("%d", &numero);
  }

  printf("Resultado da soma: %d", soma);

  return 0;
}

1.5. Operadores de incremento e decremento

O operador ++ adiciona 1 ao valor de uma variável: 

int x = 2;

x++;
printf("%d", x);

O operador -- subtrai 1 do valor de uma variável: 

int x = 5;

x--;
printf("%d", x);

Os operadores também podem ser usados em outras expressões: 

int x = 2;

printf("%d", x++);

Dentro de outras expressões, a ordem do operador define o valor que será usado:

Operador Exemplo Usado na expressão
++ ++x x + 1
++ x++ x
-- --x x - 1
-- x-- x

Lembre-se, o que vier primeiro determina o valor usado: 

int x = 2;

printf("%d", ++x);
